# Portfolio Website

## Overview

A responsive personal portfolio website showcasing my skills, projects, and professional journey. This website demonstrates modern web development practices with a clean, modularized code structure.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and intuitive user interface with smooth interactions
- **Modular Structure**: Well-organized codebase with separate directories for HTML, CSS, and JavaScript
- **Interactive Elements**: Mobile-friendly navigation menu with smooth transitions
- **Project Showcase**: Dedicated projects page with detailed project cards
- **Professional Resume**: Comprehensive resume page with skills and experience

## Pages

### 1. Home Page (index.html)
- Profile photo with hover effect
- Personal introduction and summary
- Quick statistics section
- Call-to-action buttons

### 2. About Page (pages/about.html)
- Background and qualifications
- Educational history
- Technical skills and certifications
- Conducted activities (Hour of Code, Robotics, Hackathons, Workshops)

### 3. Projects Page (pages/projects.html)
- **Project 1**: Web Application - Full-stack development showcase
- **Project 2**: Hour of Code - Interactive coding tutorial platform
- **Project 3**: Robotics - Autonomous line-following robot
- **Project 4**: Task Manager App - Cross-platform mobile application

### 4. Resume Page (pages/resume.html)
- Work experience
- Education history
- Technical skills with proficiency levels
- Certifications
- Contact information

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Font Awesome**: Icon library
- **Responsive Design**: Mobile-first approach

## Design Principles

1. **Mobile-First**: Designed for mobile devices first, then scaled up
2. **Accessibility**: Semantic HTML and ARIA labels
3. **Performance**: Optimized assets and minimal dependencies
4. **Maintainability**: Clean, well-commented code
5. **User Experience**: Intuitive navigation and smooth interactions

## Setup Instructions

1. Clone or download this repository
2. Open index.html in a web browser
3. Navigate through pages using the navigation menu
