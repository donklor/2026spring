# My Portfolio Website

This portfolio website presents my personal profile, education, skills, projects, and resume in a responsive, modern layout.

## Website Structure

- `index.html` - Home page with profile photo, introduction, and quick stats.
- `pages/about.html` - About page with background, qualifications, technical skills, and activities.
- `pages/projects.html` - Projects page featuring Project 1, Project 2 (Hour of Code), Project 3 (Robotics), and an optional Project 4.
- `pages/resume.html` - Resume page with experience, education, skills, certifications, and contact information.
- `css/style.css` - Central stylesheet with responsive layout and shared design rules.
- `js/main.js` - JavaScript for mobile navigation, active navigation highlighting, and scroll animation.

## Key Features

- Responsive navigation bar and mobile menu
- Footer with quick links and social profiles
- Public GitHub repository link and Codecademy profile link
- Cleanly organized folder structure for pages, styles, scripts, and images
- Consistent, responsive design across desktop and mobile screens

## Pages Included

1. **Home Page** - Profile photo, introduction, hero section, and action buttons.
2. **About Page** - Background, education, skills, and activity highlights including Hour of Code and Robotics.
3. **Projects Page** - Project entries with titles, descriptions, screenshots, and links.
4. **Resume Page** - Professional experience, education, skills, certifications, and contact details.

## Usage

1. Open `index.html` in a web browser.
2. Use the navigation menu to explore the About, Projects, and Resume pages.
3. Follow the GitHub and Codecademy links for additional portfolio and learning details.
